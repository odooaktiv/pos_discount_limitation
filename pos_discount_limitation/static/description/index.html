Pos Discount Limit

This module allows the user to restrict when discount is applied.

Prerequisites before Installation

No prerequisites required before installation.

Configuration

Go to Point of Sale > Configuration > Pos Discount Limit

Set Discount limit for Pos Users

Create User

Go to Settings > Users > Create a user with Point of Sale Manager/User.

Open Pos UI

Add product to cart and apply discount.

Discount Exceed Warning

Warning is raised when Point of sale user enters discount more than 50%.
